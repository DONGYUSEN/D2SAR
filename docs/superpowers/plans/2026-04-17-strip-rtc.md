> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the separate Tianyi/Lutan process scripts with a single `scripts/strip_rtc.py` entrypoint that reads everything from `manifest.json`, prefers GPU honestly, and falls back to CPU.

**Architecture:** Add a unified manifest-driven processor with small pure helper functions for sensor detection, backend selection, and output naming. Keep the actual strip implementation on the existing stable CPU path, attempt GPU first only through an explicit selection layer, and remove the old process entrypoints entirely.

**Tech Stack:** Python, GDAL, h5py, ISCE3, unittest
